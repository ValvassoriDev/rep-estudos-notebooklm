## Projeto de Estudo — Modelos de Linguagem e Engenharia de Prompt

## Contexto e Objetivos

Este projeto foi desenvolvido para o desafio da DIO com o objetivo de aprender mais sobre Inteligência Artificial e como ela pode ajudar nos estudos e no desenvolvimento de software.

O tema escolhido foi **Modelos de Linguagem e Engenharia de Prompt**, com foco em entender como funcionam os LLMs, tokens, janela de contexto e como criar prompts melhores para obter respostas mais úteis.

### Objetivos

* Entender como modelos de IA funcionam
* Aprender o que são tokens
* Compreender janela de contexto
* Melhorar a criação de prompts
* Criar um material simples para futuras revisões

## Miniguia de Estudo

### O que são LLMs

São modelos de IA treinados com muitos textos para entender e gerar linguagem natural.

Eles ajudam em respostas, resumos, traduções e até programação.

---

### O que são Tokens

Tokens são pequenas partes do texto que o modelo usa para entender informações.

Podem ser palavras, partes de palavras ou símbolos.

---

### Janela de Contexto

É o limite de tokens que o modelo consegue lembrar durante uma conversa.

Quando esse limite acaba, as informações mais antigas são removidas.
